# Система детекции ников

Проект для детекции запрещенных слов в никнеймах пользователей.

## Архитектура

Проект состоит из следующих компонентов:

- `mp/config.go` - конфигурация для хранения списков
- `mp/normalizer.go` - нормализация текста (приведение к нижнему регистру, замена символов)
- `mp/detector.go` - основной детектор с многоуровневой проверкой
- `mp/aho.go` - реализация алгоритма Ахо-Корасик для обхода списков
- `mp/detector_test.go` - тестирование
- `mp.detector_bench_test.go` - бенчмарки для тестирования скорости работы

## Конфигурация

Система поддерживает загрузку конфигурации из JSON файла `config.json`.

- `char_replacements` - замена цифр и спецсимволов на буквы
- `translit_map` - правила транслитерации.
- `blacklist` - черный список слов.
- `whitelist` - белый список слов.
- `root_blacklist` - черный список корней.

## Многоуровневая проверка

1. **Нормализация** - убираем и заменяем лишние символы, оставляем буквы
2. **Поиск слов из белого и черного списков в нике** - если нашли вхождения плохих слов без хороших, то баним. Если нашли вхождение и хороших, то смотрим, входит ли наше плохое слово в состав хорошего. Если входит, то всё ок, а если нет, то бан. Делаем с помощью Ахо-Корасик.
3. **Проверка по стеммеру** - выделение корня и проверка.
4. **Проверка корня в черном списке** - проверяем корень слова в черном списке корней.
